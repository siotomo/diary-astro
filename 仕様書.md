# diary-astro 仕様書

## 概要

シンプルさを重視した個人日記ブログサイト。
Markdownでコンテンツを管理し、Astroで静的サイトとして公開する。

---

## 要件

| 項目 | 内容 |
|------|------|
| コンテンツ管理 | Markdownファイル（Git管理） |
| 図解表示 | Mermaid対応 |
| 分類 | タグのみ |
| デザイン | モダン（カードUI） |
| OGP | 自動生成 |
| ダークモード | なし（ライトのみ） |
| RSS | なし |
| 検索 | なし |
| コスト | 無料（GitHub Pages） |

---

## 技術スタック

| 層 | 技術 | 理由 |
|----|------|------|
| フレームワーク | Astro | コンテンツファースト、Markdownネイティブ |
| ホスティング | GitHub Pages | 無料、Git連携 |
| スタイル | Tailwind CSS | ユーティリティファースト |
| Mermaid | rehype-mermaid | ビルド時SVG変換 |
| OGP | satori | タイトルから自動生成 |
